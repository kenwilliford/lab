## Context

Building a top-down solar system visualization with ambient sound for a dinner party demo.
Guests can request features via voice -> Claude Code session adds to spec.md.
This ralph-loop implements features in real-time.

## Key Constraints

- Static HTML/CSS/JS only (GitHub Pages)
- Single file preferred (solar/index.html)
- Must look good on large OLED TV
- Sound via Web Audio API with toggle button
- F11 hint for fullscreen
